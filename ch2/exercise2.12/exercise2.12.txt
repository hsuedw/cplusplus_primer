(a) int double = 3.14;
    double is C++ keyword. It cannot not be a variable name.

(b) int _;
    _ is a valid name for a variable.

(c) int catch-22;
    - cannot be part of a variable name.

(d) int 1_or_2 = 1;
    An identifier should begin with an English letter or a underscore.

(e) double Double = 3.14;
    Beause identifiers are case-sensitive, Double is a valid variable name.
